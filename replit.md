# بوت عقيدة 🤖✨

بوت تيليجرام شخصي يعمل كذكاء اصطناعي بدور ابنة عائلية اسمها "عقيدة"، مبني بـ Python ومدعوم بنموذج Llama عبر Groq.

## كيفية التشغيل
```bash
python bot.py
```

## المتغيرات البيئية المطلوبة
| المتغير | الوصف |
|---|---|
| `TELEGRAM_TOKEN` | توكن البوت من BotFather |
| `GROQ_API_KEY` | مفتاح API من Groq Cloud |

## التقنيات المستخدمة
- **Python 3.12**
- **python-telegram-bot** — مكتبة التيليجرام
- **groq** — نموذج Llama للذكاء الاصطناعي

## ملاحظات
- البوت يرد فقط على معرّفَي تيليجرام المحددَين في الكود (عبد الرحمن وحنين)
- يحفظ ذاكرة المحادثة لآخر 20 رسالة لكل مستخدم
- النموذج المستخدم: `llama-3.1-8b-instant`

## تفضيلات المستخدم
- التواصل باللغة العربية
