# 🤖 بوت الذكاء الاصطناعي الفائق بالدارجة الجزائرية الراقية

بوت تيليجرام تفاعلي متطور مدعوم بأحدث نماذج الذكاء الاصطناعي (Llama 3.3)، تم تصميمه بعناية ليتحدث بالكامل باللهجة الجزائرية (الدارجة الدزايرية) بطلاقة وعفوية تامة وبشخصية امرأة مثقفة، هادئة، ومحفزة ترفع المعنويات وتوفر النصح والذكاء لرواد البوت.

---

## ✨ الميزات الرئيسية
- **رسالة ترحيبية راقية ومشجعة**: تبدأ المحادثة بعبارات ترفع الهمم وتدعو للتوكل والعزيمة بنكهة جزائرية دافئة.
- **ذكاء اصطناعي مطور وموجه**: مدعوم بنموذج `llama-3.3-70b-versatile` من Groq للإجابة الذكية والسريعة على الأقسام العلمية والشرعية والعامة.
- **لوحة تحكم إدارية مشتركة**: دعم مدمج لإدارة البوت من قِبل أكثر من مشرف (Admin) عبر أزرار إدارية مخصصة.
- **حفظ دائم وقاعدة بيانات SQLite**: نظام مدمج يسجل بيانات المشتركين تلقائياً ويضمن عدم ضياعهم عند تحديث الكود أو إعادة تشغيل الاستضافة.
- **بث الرسائل والمنشورات**: تتيح للمشرفين تعميم المنشورات التوعوية أو الإعلانية لجميع المشتركين بضغطة زر واحدة.

---

## 🛠️ التقنيات والمكتبات المستخدمة
- **لغة البرمجة**: Python 3.x
- **مكتبة البوت**: `pyTelegramBotAPI` (Telebot)
- **محرك الذكاء الاصطناعي**: `groq` SDK
- **قاعدة البيانات**: `sqlite3` (مدمجة)

---

## 🚀 كيفية التثبيت والتشغيل المحلي

### 1. تثبيت المكتبات المطلوبة
قم بفتح الطرفية (Terminal) واكتب الأمر التالي لتثبيت المكتبات اللازمة:
```bash
pip install pyTelegramBotAPI groq
```

### 2. إعداد متغيرات البيئة (Environment Variables)
يجب عليك ضبط المتغيرات التالية في نظام تشغيلك أو داخل منصة الاستضافة (Railway):

| المتغير | الوصف | الإلزامية |
| :--- | :--- | :--- |
| `BOT_TOKEN` | توكن البوت الخاص بك من BotFather | إلزامي |
| `GROQ_API_KEY` | مفتاح تشغيل الذكاء الاصطناعي المحدث من Groq Cloud | إلزامي |
| `GROQ_MODEL` | اسم النموذج المستخدم (الافتراضي: `llama-3.3-70b-versatile`) | اختياري |

### 3. التشغيل
قم بتشغيل ملف البوت الرئيسي (بفرض أن اسمه `bot.py`):
```bash
python bot.py
```

## 🎯 أقسام البوت التفاعلية
1. **🕌 سؤال شرعي**: يوجه الذكاء الاصطناعي للإجابة الفقهية الرصينة بناءً على الكتاب والسنة بأسلوب مبسط.
2. **🔬 سؤال علمي**: لتقديم شروحات ثقافية وعلمية دقيقة وتسهيل فهم المعارف العامة للمستخدمين.
3. **✨ لوحة الإشراف**: تمكن الإدارة من معرفة عدد المشتركين الإجمالي، سحب بياناتهم التفصيلية، وعمل إذاعة للمنشورات الموحدة.
