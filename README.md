# 🤖 بوت تيليجرام الذكي باستخدام Groq (Llama 3)

بوت تيليجرام بسيط وسريع جداً مدعوم بالذكاء الاصطناعي (AI)، يقوم باستقبال رسائل المستخدمين والإجابة على أسئلتهم فوراً باستخدام نموذج `llama3-8b-8192` عبر منصة Groq API.

---

## ✨ الميزات الرئيسية
- **إجابات ذكية وفورية**: يعتمد على نموذج `Llama 3` السريع والقوي من Meta.
- **حماية تامة للمفاتيح**: يتم جلب الرموز السرية من متغيرات البيئة (Environment Variables) دون كتابتها داخل الكود.
- **استقرار دائم**: يستخدم خاصية `infinity_polling` لضمان استمرار عمل البوت دون توقف.
- **معالجة الأخطاء**: نظام مدمج يمنع توقف البوت عند حدوث أي مشكلة مع إظهار تنبيه للمستخدم.

---

## 🛠️ التقنيات والمكتبات المستخدمة
- **لغة البرمجة**: Python 3.x
- **مكتبة البوت**: `pyTelegramBotAPI` (Telebot)
- **محرك الذكاء الاصطناعي**: `groq` SDK

---

## 🚀 كيفية التثبيت والتشغيل المحلي

### 1. تثبيت المكتبات المطلوبة
قم بفتح الطرفية (Terminal) واكتب الأمر التالي لتثبيت المكتبات اللازمة:
```bash
pip install pyTelegramBotAPI groq
```

### 2. إعداد متغيرات البيئة (Environment Variables)
يجب عليك ضبط المتغيرات التالية في نظام تشغيلك أو داخل منصة الاستضافة (مثل Railway أو Render):

| المتغير | الوصف |
| :--- | :--- |
| `BOT_TOKEN` | توكن البوت الخاص بك والذي تحصل عليه من BotFather |
| `GROQ_API_KEY` | مفتاح تشغيل الذكاء الاصطناعي من منصة Groq Cloud |

### 3. التشغيل
قم بتشغيل ملف البوت الرئيسي (بفرض أن اسمه `bot.py`):
```bash
python bot.py
```

---

## 🎯 طريقة الاستخدام
1. عند الضغط على `/start` أو `/help` سيرحب بك البوت ويخبرك بجاهزيته.
2. أرسل له أي سؤال أو نص، وسيقوم بالرد عليك مباشرة من خلال الذكاء الاصطناعي.
