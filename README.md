# 🤖 بوت الذكاء الاصطناعي بالدارجة الجزائرية اللطيفة مع لوحة تحكم إدارية

بوت تيليجرام تفاعلي متقدم مدعوم بالذكاء الاصطناعي (AI)، تم تصميمه ليتحدث بالكامل باللهجة الجزائرية (الدارجة الدزايرية) المفهومة، وبشخصية امرأة هادئة ولطيفة. يحتوي البوت على نظام إدارة متكامل وقاعدة بيانات للمشرفين المحددين.

---

## ✨ الميزات الرئيسية
- **الشخصية والهوية العاطفية**: يتحدث البوت بأسلوب هادئ ولطيف جداً باستخدام الدارجة الجزائرية والرموز التعبيرية المبهجة.
- **لوحة تحكم إدارية مشتركة**: دعم مدمج لأكثر من مشرف (Admin) تتيح لهم التحكم بالبوت عبر أزرار خاصة تحت حقل الكتابة.
- **إحصائيات وقاعدة بيانات**: استخدام قاعدة بيانات مدمجة (SQLite) لحفظ بيانات المشتركين وتواريخ انضمامهم تلقائياً.
- **نظام تتبع وإشراف**: إمكانية تتبع ومراقبة آخر الرسائل التي يكتبها المستخدمون داخل البوت لضمان جودة الخدمة.
- **بث الرسائل والمنشورات**: إمكانية إرسال وتعميم منشور موحد لجميع المشتركين في البوت بضغطة زر.
- **قسم الأشعار المدمج**: يحتوي على أزرار فرعية تفاعلية لعرض أبيات شعرية وغزلية منتقاة بالهجتين الجزائرية والخليجية.

---

## 🛠️ التقنيات والمكتبات المستخدمة
- **لغة البرمجة**: Python 3.x
- **مكتبة البوت**: `pyTelegramBotAPI` (Telebot)
- **محرك الذكاء الاصطناعي**: `groq` SDK
- **إدارة البيانات**: `sqlite3`

---

## 🚀 كيفية التثبيت والتشغيل المحلي

### 1. تثبيت المكتبات المطلوبة
قم بفتح الطرفية (Terminal) واكتب الأمر التالي لتثبيت المكتبات اللازمة:
```bash
pip install pyTelegramBotAPI groq
```

### 2. إعداد متغيرات البيئة (Environment Variables)
يجب عليك ضبط المتغيرات التالية في نظام تشغيلك أو داخل منصة الاستضافة (Railway):

| المتغير | الوصف | الإلزامية |
| :--- | :--- | :--- |
| `BOT_TOKEN` | توكن البوت الخاص بك من BotFather | إلزامي |
| `GROQ_API_KEY` | مفتاح تشغيل الذكاء الاصطناعي من Groq Cloud | إلزامي |
| `GROQ_MODEL` | اسم النموذج المستخدم (الافتراضي: `llama-3.1-8b-instant`) | اختياري |

### 3. التشغيل
قم بتشغيل ملف البوت الرئيسي (بفرض أن اسمه `bot.py`):
```bash
python bot.py
```

---

## 🎯 طريقة الاستخدام والإشراف
1. **للمشترك العادي**: عند إرسال أمر `/start` يرحب به البوت بالدارجة الجزائرية، ويتلقى إجابات هادئة ولطيفة مدعومة بالذكاء الاصطناعي.
2. **للمشرفين المحددين**: تظهر لهم تلقائياً أزرار التحكم الأربعة في لوحة التوجيه (عدد المشتركين، إرسال منشور، قسم الأشعار، وتتبع الرسائل) بمجرد تفعيل معرفاتهم داخل الكود.
