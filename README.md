# 🤖 بوت الذكاء الاصطناعي بالدارجة الجزائرية اللطيفة

بوت تيليجرام تفاعلي وسريع جداً مدعوم بالذكاء الاصطناعي (AI)، تم تصميمه ليتحدث بالكامل باللهجة الجزائرية (الدارجة الدزايرية) المفهومة، وبشخصية امرأة هادئة، لطيفة، وقريبة من القلب لتقديم المساعدة والإجابة على الاستفسارات.

---

## ✨ الميزات الرئيسية
- **الشخصية والهوية العاطفية**: يتحدث البوت بأسلوب هادئ ولطيف جداً باستخدام الدارجة الجزائرية السلسة والرموز التعبيرية المبهجة.
- **سرعة فائقة في الردود**: مدعوم بنماذج شركة Groq المتطورة والسريعة (مثل `Llama 3.1`).
- **حماية تامة للمفاتيح السرية**: يتم جلب الرموز الحساسة عبر متغيرات البيئة (Environment Variables) لضمان أمان المشروع.
- **مرونة برمجية كاملة**: إمكانية تغيير وتحديث نموذج الذكاء الاصطناعي مباشرة من لوحة تحكم الاستضافة دون الحاجة لتعديل الكود.
- **استقرار دائم**: مدعوم بخاصية `infinity_polling` لضمان بقاء البوت متصلاً بالإنترنت ودون توقف.

---

## 🛠️ التقنيات والمكتبات المستخدمة
- **لغة البرمجة**: Python 3.x
- **مكتبة البوت**: `pyTelegramBotAPI` (Telebot)
- **محرك الذكاء الاصطناعي**: `groq` SDK

---

## 🚀 كيفية التثبيت والتشغيل المحلي

### 1. تثبيت المكتبات المطلوبة
قم بفتح الطرفية (Terminal) واكتب الأمر التالي لتثبيت المكتبات اللازمة:
```bash
pip install pyTelegramBotAPI groq
```

### 2. إعداد متغيرات البيئة (Environment Variables)
يجب عليك ضبط المتغيرات التالية في نظام تشغيلك أو داخل منصة الاستضافة (مثل Railway):

| المتغير | الوصف | الإلزامية |
| :--- | :--- | :--- |
| `BOT_TOKEN` | توكن البوت الخاص بك من BotFather | إلزامي |
| `GROQ_API_KEY` | مفتاح تشغيل الذكاء الاصطناعي من Groq Cloud | إلزامي |
| `GROQ_MODEL` | اسم النموذج المستخدم (الافتراضي: `llama-3.1-8b-instant`) | اختياري |

### 3. التشغيل
قم بتشغيل ملف البوت الرئيسي (بفرض أن اسمه `bot.py`):
```bash
python bot.py
```
